# Globentix CRM Dashboard

A modern, responsive Customer Relationship Management (CRM) dashboard designed for sales teams to manage leads, track conversions, and monitor key performance metrics. Built for Globentix Technology Company with a clean, professional interface and real-time data visualization.

## Features

### Dashboard & Analytics
- **KPI Metrics Overview**: Track total leads, positive responses, conversion rates, and active deals
- **Lead Conversion Funnel**: Visual representation of lead progression through sales stages
- **Real-time Charts**: Interactive data visualizations using Chart.js
- **Performance Trends**: Compare current period metrics with historical data

### Lead Management
- **All Leads**: Comprehensive view of all leads with filtering and sorting
- **Show Leads**: Specialized tracking for event/exhibition-based leads
- **Industry Leads**: Categorized industry-specific lead tracking
- **Lead Details**: Capture email, company info, show details, attendee counts, and more

### User Features
- **User Profile Management**: Update personal information and avatar
- **Settings Panel**: Configure account, security, and notification preferences
- **Announcements**: Stay updated with team notifications and system messages
- **Role-Based Interface**: Customized views based on user role (Sales Manager, Executive, Admin)

### UI/UX
- **Responsive Design**: Works seamlessly across desktop, tablet, and mobile devices
- **Modern Sidebar Navigation**: Collapsible menu with dropdown support
- **Toast Notifications**: Non-intrusive feedback for user actions
- **Modal Forms**: Clean, organized forms for adding and editing leads

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with Poppins font family
- **Charts**: Chart.js for data visualization
- **Icons**: Custom SVG icons
- **Design Pattern**: Component-based architecture
